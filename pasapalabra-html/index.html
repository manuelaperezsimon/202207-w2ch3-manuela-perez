PASAPALABRA

El juego consiste en acertar las veintisiete palabras, cada una de las cuales se corresponde con una letra del rosco para la que se ofrece una definición relativa a un concepto.

Los aciertos se reflejarán en las letras mediante el color verde, mientras que los fallos se mostrarán en color rojo y el tiempo se pausará. Para seguir con el juego, deberá clicar en el botón reanudar.

Las respuestas que se le dan el "pasapalabra" se mostrarán en color amarillo y pasará lo mismo con los errores en cuanto a reanudar el tiempo.

Para completar el rosco se dispone de un tiempo determinado: 300segundos.

IMPORTANTE: si hay palabras que se escriben con tilde, debe obviar de ponérselos.
